## Working Locally

  Clone the repository.

    $ git clone git://github.com/kraih/mojolicious.org.git
    $ cd mojolicious.org

  Install [Mojolicious](http://mojolicious.org).

    $ curl -L https://cpanmin.us | perl - -M https://cpan.metacpan.org -n Mojolicious

  Start hacking.

    $ morbo mojolicious.pl
